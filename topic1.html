How HTTP Works

What is HTTP?

HTTP (Hypertext Transfer Protocol) is the protocol that allows your web browser to communicate with servers and request information to display webpages.

HTTP Request and Response

Every time you enter a URL (like www.google.com), your browser sends an HTTP request. The server then responds with an HTTP response, which contains the webpage data your browser displays.
